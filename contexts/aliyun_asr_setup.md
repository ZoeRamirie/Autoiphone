# 阿里云实时语音识别 (流式 ASR) 准备指南

为了实现毫秒级的真人大模型对话打断体验，我们需要调用阿里云的**“智能语音交互”**服务。

请您按照以下步骤，从您的阿里云控制台获取两个核心凭据：

### 1. 登录并开通服务
1. 打开浏览器登录阿里云控制台。
2. 在左上角搜索框搜索 **“智能语音交互”**，点击进入控制台。
3. 如果尚未开通，请点击免费开通（开通后会赠送 3 个月的免费测试额度，足够跑完测试了）。

### 2. 创建一个项目并获取 Appkey
1. 在“智能语音交互”控制台的左侧菜单，点击 **“全部项目” -> “创建项目”**。
2. 项目名称随便填（比如 `AutoiphoneDemo`）。
3. 场景选择：**“语音识别” -> “实时语音识别”**。
4. 模型选择：推荐选择 **“中文普通话 - 泛热词模型”** 或最新的底座模型。
5. 创建完毕后，您会在项目列表里看到这个项目。请复制项目卡片上的 **`Appkey`** 备用（它通常是一串像 `xxxxxx` 的英文字母）。

### 3. 获取 AccessToken (访问密钥)
*注意：这不是您阿里云主账号的 RAM AK/SK，而是专属的短期 Token。*
1. 在“智能语音交互”控制台右侧或顶部，找到 **“访问口令”** 或 **“AccessToken”** 页面。
2. 点击生成，您会得到一把类似 `1234567890abcdef...` 的字符串，这就是 **`Token`**。
3. *注：手动生成的 Token 通常有 24 小时的有效期，足够我们今天测试使用。等稍后我们在正式代码里，我会写代码让它自动用您的永久 AK 自动换取 Token。*
